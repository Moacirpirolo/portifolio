# Gael Burger

Site estático para hamburgueria artesanal com cardápio digital, carrinho e envio de pedido para WhatsApp.

## Como abrir

Abra `index.html` no navegador.

## Ajustes rápidos

- WhatsApp: edite `script.js` e troque `5511999999999` pelo número real com DDI e DDD.
- Endereço e horários: edite a faixa de informações em `index.html`.
- Itens do cardápio: edite o array `products` em `script.js`.
- Fotos: usam URLs externas do Unsplash e precisam de internet para carregar.
