# Lili do Cambuci

Site estático para venda artesanal de produtos de cambuci, com loja digital, cestinha e envio para WhatsApp.

## Como abrir

Abra `index.html` no navegador.

## Ajustes rápidos

- WhatsApp: edite `script.js` e troque `5511999999999` pelo número real com DDI e DDD.
- Produtos: edite o array `products` em `script.js`.
- Endereço e entrega: edite as seções de texto em `index.html`.
- Fotos externas: algumas imagens vêm do Unsplash e precisam de internet para carregar.
